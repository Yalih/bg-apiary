# Dashboard Apiaries 2.1

## Cel

Pulpit pokazuje pasieki jako główne miejsca pracy, a nie pojedyncze ule.

## Karta pasieki

Pokazuje:

- nazwę,
- lokalizację,
- liczbę rodzin,
- pogodę dla tej pasieki,
- aktualny pożytek,
- liczbę prac,
- alerty.

## Ule

Ule pozostają dostępne po wejściu w pasiekę oraz przez ekran Ule. Nie usunięto żadnej logiki uli.
