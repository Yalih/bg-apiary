# BgApiary 2.1 Premium Dashboard

## Cel

Przebudowa wyłącznie ekranu Pulpit bez dodawania nowych funkcji i bez zmian logiki.

## Zasada

Maksimum informacji, minimum tekstu.

## Sekcje

- Hero,
- Pogoda,
- Pożytek,
- Rekomendacja dnia,
- Moje ule,
- Najbliższe prace,
- Alerty,
- Asystent.

## Co zmieniono

- skrócono Pulpit,
- powiększono karty i grafiki,
- ograniczono liczbę widocznych uli do 2,
- ograniczono najbliższe prace do 2,
- uproszczono rekomendację dnia,
- połączono statusy w zwarte kafelki,
- poprawiono hierarchię informacji,
- dodano dedykowany CSS `premiumDashboard21.css`.

## Bez zmian

Nie zmieniono:

- modeli danych,
- backupu,
- importu/eksportu,
- logiki pasiek,
- logiki uli,
- logiki prac,
- logiki asystenta,
- localStorage.
