# UI Guidelines 2.0

## Puste stany

Nie używać:

- Dodaj pierwszy wpis,
- Dodaj pierwszy element,
- Nie udało się odnaleźć.

Używać:

- Dodaj pierwszy ul,
- Dodaj pierwszą pasiekę,
- Dodaj zadanie,
- Importuj kopię zapasową.

## Mobile

- minimum 44 px wysokości klikanych elementów,
- brak poziomego scrolla,
- krótkie opisy,
- szybkie akcje blisko kciuka.

## Premium

- kremowe tło,
- ciemna zieleń,
- złoto miodowe,
- jasne karty,
- duże zaokrąglenia,
- delikatne cienie.
