# BgApiary 2.1 Premium Visual System

## Cel

Ujednolicenie wyglądu całej aplikacji bez zmian logiki i bez zmian modeli danych.

## Tokeny

Plik:

`src/logic/premiumVisualSystem21.ts`

Zawiera:

- kolory,
- cienie,
- promienie,
- spacing,
- typografię,
- styl kart.

## Komponenty

Plik:

`src/components/premium/PremiumVisualSystem.tsx`

Komponenty:

- `PremiumCard`,
- `PremiumTile`,
- `PremiumBadge`,
- `PremiumSection`,
- `PremiumHeader`,
- `PremiumButton`,
- `PremiumEmptyState`,
- `PremiumSkeleton`.

## CSS

Plik:

`src/styles/premiumVisualSystem21.css`

Działa jako globalna warstwa wizualna dla istniejących ekranów:

- Pulpit,
- Ule,
- Szczegóły ula,
- Raporty,
- Plan,
- Pogoda,
- Pożytek,
- Więcej,
- Asystent.

## Zasady

- jedna paleta kolorów,
- jedna skala promieni,
- jedna skala cieni,
- jedna skala odstępów,
- jeden styl kart,
- czerwony tylko dla alarmów,
- złoto dla CTA i wyróżnień,
- butelkowa zieleń jako główny kolor.
