# Nectar Accuracy 2.1

## Cel

Pożytek jest wyliczany lokalnie z daty, kalendarza sezonowego, danych pasieki i pogody.

## Zasady

- Aplikacja nie udaje globalnego API pożytków.
- Ręcznie wpisany pożytek pasieki ma pierwszeństwo.
- Kalendarz lokalny obejmuje: sad, mniszek, rzepak, akację, lipę, facelię, grykę, nawłoć, wrzos i spadź.
- Pogoda może osłabić lub wzmocnić ocenę pożytku.
- Wynik zawiera fazę: start, pełnia, końcówka lub brak.
