# Visual Asset Guide

## Hero

Do szczegółów ula używać wyłącznie `HiveHeroImage`.

## Karty

Do kart uli używać `HiveCardImage`.

## Miniatury

Do małych widoków używać `HiveMiniImage`.

## Statusy

Status to nakładka `HiveStatusOverlay`, nie osobny ul.

## Zakaz

Nie używać w UI:

- pełnych plansz,
- sprite sheetów,
- tymczasowy opis technicznyów,
- emoji jako ikon modułów,
- bezpośrednich ścieżek do assetów poza mappingiem.
