# BgApiary 2.0 BGS Theme

## Cel

Wdrożenie motywu inspirowanego grafiką Pasieka BGS.

## Elementy referencji

- kremowe tło,
- butelkowa zieleń,
- złoto jako akcent,
- duże zaokrąglone karty,
- Pulpit z krótkim podsumowaniem,
- karty uli z miniaturą i paskiem siły,
- dolna nawigacja z centralnym złotym plusem,
- szczegóły ula jako kafelki,
- historia jako oś czasu.

## Ograniczenia

Nie dodano nowych funkcji biznesowych. Motyw zmienia UI, layout i sposób prezentacji danych.
