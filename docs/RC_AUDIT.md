# BgApiary 1.0 RC - finalny audyt przed 1.1

## Zakres RC

Wersja RC nie dodaje nowych funkcji. To finalny etap sprawdzenia stabilności, UX, dokumentacji i gotowości projektu do przejścia na linię 1.1.

## Obszary audytu

- testy automatyczne,
- build produkcyjny,
- struktura Netlify ZIP,
- główne ścieżki użytkownika,
- localStorage,
- eksport i import JSON,
- backup,
- raporty,
- Asystent,
- zadania i kalendarz,
- zdjęcia,
- formularze,
- spójność nazw,
- spójność ikon,
- spójność kolorów,
- dostępność mobilna,
- dokumentacja.

## Główne ścieżki użytkownika

- Start → Dashboard.
- Dashboard → pilne zadanie → ul albo formularz.
- Pasieki → lista uli → szczegóły ula.
- Szczegóły ula → przegląd.
- Szczegóły ula → karmienie.
- Szczegóły ula → notatka.
- Szczegóły ula → zdjęcie.
- Szczegóły ula → historia.
- Zadania → zadanie → ul/formularz.
- Kalendarz → zadanie → ul/formularz.
- Asystent → rekomendacja → działanie.
- Asystent → rekomendacja → utwórz zadanie.
- Raporty → raport ula/pasieki/globalny.
- Backup → eksport/import.
- Więcej → raporty/backup.

## Spójność kolorów

- zielony: OK,
- żółty: obserwacja / średni priorytet,
- pomarańczowy: wysoka uwaga,
- czerwony: pilne / krytyczne,
- niebieski: informacja / analiza,
- ciemny: główna akcja.

## Wynik

RC jest gotowe do przejścia na etap 1.1, jeżeli:

- testy przechodzą,
- build przechodzi,
- Netlify ZIP ma poprawną strukturę,
- eksport/import działa,
- localStorage działa,
- dokumentacja jest aktualna,
- nie dodano nowych funkcji.
