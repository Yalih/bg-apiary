# BG Apiary Premium 2.0

## Cel

Dopasowanie aplikacji do referencji BG APIARY: nowoczesny, spokojny, premium interfejs dla pszczelarzy.

## Najważniejsze decyzje

- mniej tekstu,
- większe karty,
- szybki dostęp do codziennych akcji,
- karta pożytku,
- karty rodzin jako główny element Pulpitu,
- zielony jako kolor zaufania,
- złoto jako kolor akcji i miodu,
- czerwony tylko dla alarmów.

## Bez zmian

Motyw nie dodaje funkcji biznesowych i nie zmienia modeli danych, backupu, localStorage ani logiki kont.
