# Apiary Edit GPS Fix 2.1

## Cel

Naprawa braku możliwości zmiany lokalizacji GPS istniejącej pasieki.

## Dodano

Na liście pasiek każda pasieka ma teraz:

- przycisk „Zmień lokalizację pasieki”,
- przycisk „Zmień lokalizację GPS”,
- ręczne pola:
  - miejscowość / opis lokalizacji,
  - szerokość geograficzna,
  - długość geograficzna,
- zapis lokalizacji do istniejącej pasieki.

## Wykorzystanie danych

Zaktualizowane `latitude` i `longitude` są dalej używane przez pogodę Open-Meteo i lokalny pożytek.

## Bez zmian

Nie zmieniono modeli biznesowych, backupu, importu/eksportu ani logiki uli.
