# BG Apiary 1.0 UI/UX Spec

## Główne zasady

- Mobile first.
- Duże przyciski.
- Czytelność w słońcu.
- Szybki dostęp do przeglądu ula.
- Pierwszy ekran ma budować zaufanie i efekt jakości.
- Dashboard ma odpowiadać na pytanie: co wymaga uwagi?

## Pierwszy ekran

Elementy:

- logo BG Apiary,
- hasło: „Twoja pasieka pod kontrolą”,
- krótki opis aplikacji,
- korzyści,
- karta logowania/rejestracji,
- nowoczesne tło z motywem plastra i miodu,
- onboarding po rejestracji.

## Statusy uli

- OK: zielony,
- Obserwuj: żółty,
- Wymaga działania: czerwony,
- Brak danych: szary.
