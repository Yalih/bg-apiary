# BG Apiary Premium Polish

## Cel

Doprowadzić motyw BG APIARY do spójnego, profesjonalnego wyglądu premium.

## Priorytety

1. Szczegóły ula.
2. Karty uli.
3. Plan prac.
4. Raporty.
5. Puste stany.
6. Ikony i mikrointerakcje.

## Szczegóły ula

Ekran składa się z:

- dużego nagłówka,
- ilustracji ula,
- paska siły,
- zakładek,
- kafelków danych,
- karty ostatniego przeglądu,
- osi czasu,
- miniatur zdjęć,
- szybkich akcji.

## Ograniczenie

Brak nowych funkcji biznesowych. Tylko profesjonalizacja UI.


## Weather & Nectar

Pogoda korzysta z Open-Meteo API bez klucza. Dane są cache'owane lokalnie. Offline pokazuje ostatnią prognozę albo fallback.

Pożytek nie korzysta z globalnego API. Jest wyliczany z lokalnego kalendarza pożytków w Planie Sezonu i pogody.
