# BgApiary 2.1 Premium Hive Experience

## Cel

Dopracowanie wizualne doświadczenia ula bez zmian logiki i bez zmian modeli danych.

## Poprawione obszary

- lista uli,
- karty uli,
- szczegóły ula,
- historia,
- hero header,
- matka,
- statusy,
- nakładki,
- timeline,
- hero illustration,
- queen dot,
- overlay,
- status badge,
- FAB,
- kafelki.

## Dynamiczne ilustracje

Ilustracje są dobierane przez istniejące dane:

- siła rodziny,
- pora roku,
- status ula.

Nie dodano nowych pól danych.

## Hero

Hero ula używa `HiveHeroImage` z AssetManagera 2.1 oraz nakładki `HiveOverlay`.

## Queen Dot

Matka korzysta z dynamicznej opalitki liczonej z roku matki.

## Timeline

Historia ula jest prezentowana jako czytelna oś czasu z ikonami zdarzeń i ograniczeniem widocznych wpisów.

## FAB

Szybkie akcje pozostają istniejące:

- Przegląd,
- Karmienie,
- Matka,
- Zdjęcie,
- Notatka.
