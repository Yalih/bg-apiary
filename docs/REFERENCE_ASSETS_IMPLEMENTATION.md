# Reference Assets Implementation

Ta wersja korzysta z grafik wyciętych z przesłanych plansz referencyjnych.

## Podpięto

- Karty uli: `BgApiaryReferenceHiveByState`
- Pulpit / Moje rodziny: `BgApiaryReferenceHiveByState`
- Szczegóły ula / hero: `BgApiaryReferenceHiveByState`
- Szczegóły ula / kafelek Matka: `BgApiaryReferenceQueenByStatus`

## Assety

- `public/assets/bgapiary-reference/hives/`
- `public/assets/bgapiary-reference/queens/`
- `public/assets/bgapiary-reference/status/`

## Źródło

Grafiki są wycięte z przesłanych plansz. Nie generowano kolejnych grafik.
