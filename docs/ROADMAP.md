# BG Apiary roadmap

## Sprint 1 - project cleanup

Goal: prepare the repository for predictable development.

Scope:

- remove local Git history from shared archives,
- add `.gitignore`,
- refresh `README.md`,
- add documentation index,
- add architecture overview,
- add Sprint 1 cleanup note,
- keep application logic unchanged.

Status: done.

## Sprint 2 - UI and branding polish

Goal: make the first screen feel like a polished commercial product.

Scope:

- dashboard polish,
- top bar and version label cleanup,
- favicon and metadata check,
- mobile visual polish,
- reusable visual tokens.

## Sprint 3 - hive management module

Goal: make hive management clearer and more scalable.

Scope:

- hive list structure,
- hive details cleanup,
- status indicators,
- frame count and colony strength presentation,
- reusable hive components.

## Sprint 4 - inspection notes

Goal: improve the inspection workflow.

Scope:

- inspection form structure,
- queen status,
- brood stages,
- food amount,
- frame notes,
- next action suggestion.

## Sprint 5 - apiary task planner

Goal: add practical daily planning.

Scope:

- feeding tasks,
- inspection tasks,
- treatment tasks,
- reminders and priority labels.

## Sprint 6 - backend foundation

Goal: move from local-only data toward a real multi-device product.

Scope:

- API foundation,
- PostgreSQL,
- environment configuration,
- production deployment plan.

## Sprint 7 - authentication

Goal: add user accounts.

Scope:

- admin user,
- login,
- session handling,
- protected views.

## Sprint 8 - backup and production safety

Goal: protect production data and deployment.

Scope:

- database backups,
- `.env` hardening,
- server notes,
- recovery procedure.

## Sprint 9 - PWA and mobile experience

Goal: improve installable mobile experience.

Scope:

- manifest polish,
- app icons,
- splash screen,
- offline behavior review.

## Sprint 10 - BG Apiary 1.0

Goal: first stable production release.

Scope:

- release candidate tests,
- changelog,
- tag `v1.0.0`,
- production checklist.
